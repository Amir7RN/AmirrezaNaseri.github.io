About Me

I am a Ph.D. candidate in Biomedical Engineering at NC State/UNC focusing on the design and control of wearable robotic systems such as prostheses and exoskeletons. My mission is to translate research into impactful products that improve human mobility and independence.

Projects

  • Mechanical Design and System Integration of Energy-Efficient Mechanisms for Wearable Assistive Devices
    This project showcases the complete development of a passive hybrid ankle prosthesis that replicates biological ankle behavior through a compact mechanical system.

    The workflow included:

    • Biomechanical analysis of human ankle motion
    • Conceptual mechanism simulation in MATLAB
    • 3D modeling in CATIA/SolidWorks
    • Stress analysis and structural verification
    • Fabrication drawing and machining
    • Human subject testing (able-bodied and amputee)

  • Real-Time Adaptive Control of Actuated Robotic Systems Using Force Feedback — Designed and validated a bioinspired controller that uses vertical ground reaction force (GRF) feedback to modulate ankle impedance, enabling stable and human-like locomotion in simulation and prosthetic experiments.

    Workflow Summary:

    1. Analyzed human-like ankle torque generation using GRF-modulated impedance functions.
    2. Developed simulation in Simulink with neuromuscular model and FMCA control.
    3. Designed finite-state machine for phase-dependent ankle control (see figure below).
    4. Tested on powered prosthetic foot with real-time GRF feedback and compared to reference data.

  • Robotic Ankle Prosthesis with Shared Autonomy — Developed a control framework for a powered ankle prosthesis using real-time motor control and sensor fusion to enable adaptive assistance during obstacle avoidance.
    • Programmed and tuned ELMO Gold Solo Twitter servo drives for BLDC motors in position and torque control modes.
    • Designed a real-time trajectory tracking controller based on normative gait data, implemented in TwinCAT3–Simulink RT environment.
    • Integrated EMG and IMU sensors with fast filtering and threshold logic to detect user intent deviations from normal walking.
    • Implemented control switching logic: transitions from position control to torque augmentation when obstacle engagement is detected.
    • Configured sensor calibration pipelines for load cells, potentiometers, and multi-turn encoders on a bypass-enabled prosthetic system for AB subject testing.

    Workflow Summary:

    1. Tune motor using ELMO studio for accurate position tracking of normative ankle angles.
    2. Detect user intent via EMG signal thresholding; determine phase and disturbance onset.
    3. Trigger torque boost to help navigate obstacles when abnormal muscle contraction is sensed.
    4. Verify closed-loop control in experimental setup using physical obstacle interaction and bypass prosthetic fixture.
