Sensor-to-EtherCAT Interface via Raspberry Pi

Developed a custom data pipeline enabling IMU data transmission from a UART-based sensor to an EtherCAT-based control system using Raspberry Pi and a LAN9252 EtherCAT slave module.

Workflow Summary:

  1. Read real-time IMU data via UART using a Raspberry Pi.
  2. Processed and buffered sensor data using custom Python & C++ scripts.
  3. Transmitted formatted data to the LAN9252 EtherCAT slave over SPI protocol using custom C scripts.
  4. Successfully integrated with TwinCAT master for real-time robotic control applications.
  5. Designed and tested the entire system using TwinCAT diagnostics for signal integrity and timing validation.

Adaptive AI-Driven Control with Fault-Tolerance for Robotic Actuation Systems

Developed and validated a machine-learning-based controller for a robotic actuated knee prosthesis, augmented with an active safety mechanism (Fault-Tolerance Mechanism; FTM) to detect and compensate for internal control faults in real time, enhancing user safety, balance, and gait stability.

Workflow included:

  1. Real-time controller development using TwinCAT3–Simulink RT over EtherCAT.
  2. Upgraded control loop frequency from 100 Hz (LabView) to 1 kHz (TwinCAT) for responsive adaptation.
  3. Designed and integrated full embedded system architecture including KEB EtherCAT bus, NI DAQ interface, Raspberry Pi, and SPI-to-EtherCAT slave modules.
  4. Augmented the controller with higher-level physics-informed, machine-learning-driven FTM for mitigating intrinsic faults.
  5. Experimental validation with simulated faults during treadmill walking trials with human subjects (Motion Capture).
  6. Quantitative and subjective evaluation of gait disturbances and stability (Biomechanics).

The closed-loop controller drives the motor based on user interaction.

Adaptive Mode-Switching Control with Shared Autonomy for Robotic Actuation

Designed and implemented a control architecture for a wearable robotic system (case study: robotic ankle prosthesis) featuring dual-mode operation, position control for autonomous trajectory tracking, and torque control based on volitional physiological signals (EMG) for adaptive user-driven behavior. The system enables seamless transitions to assist users during dynamic tasks like obstacle avoidance.

Workflow Summary:

  1. Programmed and tuned ELMO Gold Solo Twitter servo drives for BLDC motors operating in both position and torque control modes
  2. Integrated real-time EMG and IMU sensor data using fast signal conditioning and threshold-based decision logic
  3. Implemented hybrid control mode switching from trajectory tracking to torque augmentation based on user intent deviation
  4. Calibrated load cells, potentiometers, and multi-turn encoders for accurate sensor feedback on a bypass-ready prosthesis tested with AB subjects

System overview illustrating subject testing, simulation-based motion analysis, and the physical setup using bypass-enabled prosthetic hardware.

Demonstration of shared autonomy: an able-bodied subject walks with a robotic ankle prosthesis in position control mode. Upon detecting an approaching obstacle, volitional muscle activation triggers torque control, enhancing foot clearance for successful obstacle negotiation.

Physics-Informed Control of Wearable Robotic Systems: Hip Exoskeleton Case Study

Designed and implemented a real-time controller for a frontal-plane actuated robotic hip exoskeleton that assists balance by regulating whole-body angular momentum. The exoskeleton applies adaptive adduction torques based on excess torso momentum during narrow beam walking, improving stability in highly constrained environments.

Workflow Summary:

  1. Developed a physics-based assistance strategy using simplified whole-body human motion analysis (MATLAB)
  2. Implemented a momentum-based controller with admittance control in Simulink and deployed it in real time using TwinCAT at 1 KHz (C++)
  3. Developed the required embedded system architecture for fusion sensor integration and real-time control execution
  4. Conducted validation through narrow beam walking experiments with able-bodied participants to assess balance augmentation

Real-Time Adaptive Control of Actuated Robotic Systems Using Force Feedback

Designed and validated a bioinspired controller that uses vertical ground reaction force (GRF) feedback to modulate ankle impedance, enabling stable and human-like locomotion in simulation and physical testing. Implemented Force Modulated Control Ankle (FMCA) control on a robotic prosthetic ankle system.

Workflow Summary:

  1. Analyzed human-like ankle torque generation using GRF-modulated impedance functions (MATLAB).
  2. Developed simulation in Simulink with a neuromuscular model (Geyer et al., 2016) and FMCA control.
  3. Designed an adaptive impedance-based finite-state machine for phase-dependent ankle control, implemented using xPC Target and Simulink Real-Time at 1 kHz.
  4. Tested on a powered prosthetic foot with real-time GRF feedback and evaluated performance using motion capture and indirect respirometry.

Quantitative Balance Metrics for Robust Locomotion and Disturbance Sensitivity Analysis

This project investigates the sensitivity of various quantitative balance metrics to internal control disturbances during bipedal locomotion. Using human-subject experiments with a powered knee prosthesis under control fault conditions, the study evaluates which metrics best capture balance impairments. The findings support the design of more effective real-time balance control systems for wearable robotic platforms.

Key Contributions:

  1. Benchmarking commonly used balance metrics (e.g., Margin of Stability, whole body angular momentum, inclination angle) against user-perceived stability in the presence of internal control disturbances of robotic prosthesis.
  2. Identifying metrics with higher sensitivity to control errors, enabling improved disturbance rejection and real-time fault detection.
  3. Translating findings to inform balance-aware controller design for assistive robotics, and can be used for humanoid locomotion control.

Comparative sensitivity analysis of multiple balance metrics under normal and faulty control conditions during treadmill walking.
