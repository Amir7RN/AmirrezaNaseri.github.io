Real-Time Adaptive Control of Actuated Robotic Systems Using Force Feedback

Designed and validated a bioinspired controller that uses vertical ground reaction force (GRF) feedback to modulate ankle impedance, enabling stable and human-like locomotion in simulation and prosthetic experiments.

Workflow Summary:

  1. Analyzed human-like ankle torque generation using GRF-modulated impedance functions.
  2. Developed simulation in Simulink with neuromuscular model and FMCA control.
  3. Designed finite-state machine for phase-dependent ankle control (see figure below).
  4. Tested on powered prosthetic foot with real-time GRF feedback and compared to reference data.

Sensor-to-EtherCAT Interface via Raspberry Pi

Developed a custom data pipeline enabling IMU data transmission from a UART-based sensor to an EtherCAT-based control system using Raspberry Pi and a LAN9252 EtherCAT slave module.

Workflow Summary:

  1. Read real-time IMU data via UART using a Raspberry Pi.
  2. Processed and buffered sensor data using custom Python scripts.
  3. Transmitted formatted data to the LAN9252 EtherCAT slave over SPI protocol.
  4. Successfully integrated with TwinCAT master for real-time robotic control applications.
  5. Designed and tested the entire system using oscilloscope and TwinCAT diagnostics for signal integrity and timing validation.

Robotic Ankle Prosthesis with Shared Autonomy

Developed a control framework for a powered ankle prosthesis using real-time motor control and sensor fusion to enable adaptive assistance during obstacle avoidance.

Workflow Summary:

  1. Programmed and tuned ELMO Gold Solo Twitter servo drives for BLDC motors in position and torque control modes.
  2. Integrated EMG and IMU sensors with fast filtering and threshold logic to detect user intent deviations from normal walking.
  3. Implemented control switching logic: transitions from position control to torque augmentation when obstacle engagement is detected.
  4. Configured sensor calibration pipelines for load cells, potentiometers, and multi-turn encoders on a bypass-enabled prosthetic system for AB subject testing.

Subplots show simulated EMG environment, real-time controller, and prosthetic hardware used for AB testing

Robotic Knee Prosthesis with Fault-Tolerant AI Control

Developed and validated a machine-learning-based active Fault-Tolerance Mechanism (FTM) for a robotic knee prosthesis to detect and compensate for internal control faults in real time, improving balance, gait stability, and user safety.

Workflow included:

  1. Real-time controller development using TwinCAT3–Simulink RT over EtherCAT
  2. Upgraded control loop frequency from 100 Hz to 1 kHz for responsive gait adaptation
  3. Experimental validation with simulated faults in human subject treadmill walking
  4. Quantitative and subjective evaluation of gait disturbances and stability
