Sensor-to-EtherCAT Interface via Raspberry Pi

Developed a custom data pipeline enabling IMU data transmission from a UART-based sensor to an EtherCAT-based control system using Raspberry Pi and a LAN9252 EtherCAT slave module.

Workflow Summary:

  1. Read real-time IMU data via UART using a Raspberry Pi.
  2. Processed and buffered sensor data using custom Python & C++ scripts.
  3. Transmitted formatted data to the LAN9252 EtherCAT slave over SPI protocol using custom C scripts.
  4. Successfully integrated with TwinCAT master for real-time robotic control applications.
  5. Designed and tested the entire system using TwinCAT diagnostics for signal integrity and timing validation.

Adaptive AI-Driven Control with Fault-Tolerance for Robotic Actuation Systems

Developed and validated a machine-learning-based controller for a robotic actuated knee prosthesis, augmented with an active safety mechanism (Fault-Tolerance Mechanism; FTM) to detect and compensate for internal control faults in real time, enhancing user safety, balance, and gait stability.

Workflow included:

  1. Real-time controller development using TwinCAT3–Simulink RT over EtherCAT.
  2. Upgraded control loop frequency from 100 Hz (LabView) to 1 kHz (TwinCAT) for responsive adaptation.
  3. Designed and integrated full embedded system architecture including KEB EtherCAT bus, NI DAQ interface, Raspberry Pi, and SPI-to-EtherCAT slave modules.
  4. Augmented the controller with higher-level physics-informed, machine-learning-driven FTM for mitigating intrinsic faults.
  5. Experimental validation with simulated faults during treadmill walking trials with human subjects (Motion Capture).
  6. Quantitative and subjective evaluation of gait disturbances and stability (Biomechanics).

Robotic Ankle Prosthesis with Shared Autonomy

Developed a control framework for a powered ankle prosthesis using real-time motor control and sensor fusion to enable adaptive assistance during obstacle avoidance.

Workflow Summary:

  1. Programmed and tuned ELMO Gold Solo Twitter servo drives for BLDC motors in position and torque control modes.
  2. Integrated EMG and IMU sensors with fast filtering and threshold logic to detect user intent deviations from normal walking.
  3. Implemented control switching logic: transitions from position control to torque augmentation when obstacle engagement is detected.
  4. Configured sensor calibration pipelines for load cells, potentiometers, and multi-turn encoders on a bypass-enabled prosthetic system for AB subject testing.

Momentum-Based Control for Hip Exoskeleton to Enhance Mediolateral Balance

Designed and implemented a real-time controller for a frontal-plane actuated hip exoskeleton that assists balance by regulating whole-body angular momentum. The exoskeleton applies adaptive ab/adduction torques based on excess torso momentum during beam walking, improving stability in highly constrained environments.

Workflow Summary:

  1. Modeled the body as a three-link system and derived frontal-plane momentum equations from IMU signals.
  2. Implemented a momentum-based controller in Simulink with real-time execution in TwinCAT at 1000 Hz.
  3. Designed partial torque assistance strategy (20% stance leg, 5% swing leg) based on torso sway thresholds.
  4. Built the hardware stack with EtherCAT modules, Raspberry Pi + Lan9252, load cells, and brushless motors.
  5. Validated the system during narrow beam walking, a high-challenge stability task with able-bodied and amputee users.
